## 6. コンテンツルール（重要）

### 6.1 外部に出さない製品名

ページは外部パートナー向けのため、以下の製品名は**使用禁止**：

| 出さない名前 | 代わりに使う表現 |
|-------------|----------------|
| FileMaker | 患者・スタッフDB / 患者DB |
| HOMIS | 電子カルテ（ブラウザベース） |
| Tukusi | AI業務自動化PF |
| CrossLog | 訪問診療管理システム |
| chromedp | ブラウザ自動化 / Go製ブラウザ自動化エンジン |
| clasp | GAS CLI |

### 6.2 法人名

- ✅ 正しい: **医療法人おひさま会**
- ❌ 間違い: 医療法人社団おひさま会

### 6.3 パートナー募集について

- パートナー募集フォーム・問い合わせ先は**掲載しない**
- 純粋な「実績紹介」に徹する
